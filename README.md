# Head Up Agency — Website

Multilingual (EN / 中文 / BM) lead-generation website for **Head Up Agency** — a digital marketing agency specialising in helping F&B businesses grow via GrabFood/Foodpanda marketing and POS system solutions.

## Quick Start

1. **Read `CLAUDE.md`** — essential rules and conventions
2. **Read `HeadUp_Agency_Brief.md`** — full design system, pages, and feature spec
3. Scaffold the Next.js app:
   ```bash
   npx create-next-app@latest . --typescript --tailwind --eslint --app
   npx shadcn@latest init
   npm install framer-motion lucide-react next-intl react-hook-form zod react-day-picker \
               next-sanity @sanity/image-url resend next-sitemap @next/third-parties
   ```
4. `npm run dev`

## Stack

Next.js 14 (App Router) · TypeScript · Tailwind · next-intl · Sanity · Framer Motion · React Hook Form + Zod · Resend · shadcn/ui
